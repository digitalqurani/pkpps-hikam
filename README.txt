README - pkpps-hikam (ready to upload)
1) Upload this ZIP to aaPanel -> File Manager -> /www/wwwroot/
   then extract to folder /www/wwwroot/pkpps-hikam
2) Edit config.php with your DB credentials (or use aaPanel to create DB and user)
3) Visit https://pkpps-hikam.mdiq.biz.id/setup.php  (run once) to create tables and default users
4) After setup, DELETE setup.php
Default accounts created by setup.php:
 - admin / admin123  (role: admin)
 - guru1 / guru123   (role: guru)
 - santri1 / santri123 (role: santri)
